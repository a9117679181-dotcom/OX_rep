# Bonus Materials Plan

Возможные бонусные материалы:
- чеклист подготовки брифа;
- шаблон структуры проекта;
- список ограничений для честной AI-коммуникации;
- набор стартовых промптов;
- чеклист проверки результата человеком.

Бонусы должны быть короткими и применимыми сразу после мини-интенсива.
