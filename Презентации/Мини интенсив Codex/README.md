# Мини-интенсив: Codex как AI-агент: быстрый старт

Первый проектный каркас мини-интенсива по практическому использованию Codex для рабочих задач.

Формат: видео без человека в кадре, screen recording, AI-озвучка и простые анимационные объяснения.

Целевая аудитория: предприниматели, владельцы малого бизнеса, фрилансеры и специалисты, которым нужен быстрый вход в работу с Codex без перегруза теорией.

Ключевая формула курса: человек задает рамки, цель и критерии качества; Codex помогает структурировать работу, готовить черновики и ускорять итерации; человек проверяет, уточняет и принимает решения.

На старте проект должен показать путь от сырой идеи к структуре проекта, сценарному плану, демонстрациям, промптам, лендинговой заготовке и рабочим материалам.

Важно не обещать заработок, замену сотрудников или автономную работу AI без участия человека.
