# Demo Plan

Основное демо: показать, как Codex превращает `source_brief.txt` в стартовый проектный каркас.

Возможные демонстрации:
- чтение исходного брифа;
- создание структуры папок;
- генерация кратких планов по разделам;
- проверка ограничений;
- фиксация того, что финальные решения остаются за человеком.

Демо должно быть простым и повторяемым, без сложной настройки и без лишней автоматизации.
